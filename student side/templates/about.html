Library Management System


It is indispensable to maintain an efficient software that records details related to library activities such as information about members, books and staff, and also payment of due fine accordingly etc. The software keeps a record of details of members, books and safely maintains the database. We have to design a software to implement Library System Management. Our aim is to design such a software to record all such information and to access them in simple way. The members can
1. Issue a Book
2. Return a Book
3. Search for a book
4. Request a book
5. Update the system library
6. Calculate fine
